See:
- `NOTICE` (required attribution)
- `TRADEMARK.md` (identity/trademark limits)
- `CITATION.cff` ("Cite this repository")
- `FORKING.md` (clean reuse checklist)

## Quick Start

1. Click **Use this template** in GitHub.
2. Create your profile repo as `<your-user>/<your-user>`.
3. Replace placeholders in:
   - `PROFILE.template.md`
   - `examples/README-minimal.md` or `examples/README-full.md`
4. Replace identity assets:
   - Name/handles/links
   - Hero/banner images and logos
5. Keep attribution and legal files in place.

## What To Replace First

- Name and headline
- Social and CTA links
- Personal/brand visuals
- Project highlights and proof points

## Template Structure

- `PROFILE.template.md`: starter with placeholders
- `components/`: reusable markdown sections (`hero`, `skills`, `projects`, `cta`)
- `examples/`: minimal/full README variants
- `docs/`: style and layout references
- `.github/`: issue templates/workflows (optional to keep)
